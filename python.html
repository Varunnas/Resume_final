SOLAR TRACKER

The aim of this project is to design and implement a light tracking system for a solar panel using an Arduino Uno board. The system will detect the direction of the maximum sunlight intensity and adjust the solar panel accordingly, maximizing its efficiency in converting sunlight into electricity.

Working Prototype First test run

With everything in place, the solar panel should now be able to adjust its angle according to the angle of sunlight hitting on it, maximizing its efficiency and output.

How we built it? B.T.S

  1. The first step in building this project is to connect the Arduino UNO to the breadboard and wire up the servo motors. We will also need to connect the light sensor and potentiometer to the breadboard, as well as the power source for the motors.
  2. Once all of the components are wired up, we can begin programming the Arduino UNO to receive input from the sensors and control the motors accordingly. This will involve writing code in the Arduino IDE and uploading it to the microcontroller.

Final Step Almost There

Finally, we can mount the solar panel onto the rotating platform and attach the servo motors to it. We will also need to calibrate the sensors and adjust the code as necessary to ensure that the solar panel is tracking the sun accurately.
